modul TDS

Author: H.Ristau

Pure javascript modul to connect MSSQL Databases over the TDS protocol. TDS version 7.0 will be used, so connections to SQL Server 2000, 2005 and 2008 are possible.

function TDS.end

closes the database connection and destroys the connection pool

 


TDS.prototype.end = function() {
  Object.keys(pools).forEach(function(name) {
    var pool = pools[name];
    pool.drain(function() {
      pool.destroyAllNow();
    });
  })
}

module.exports = new TDS(Connection);
